# LL1 CFG Parser

This project aims to parse a LL1 unambiguous CFG

### [Web Deployment](https://toc-cfg-parser.pages.dev/)

### Local Deployment

1. Clone the repo
2. Run `cargo install trunk`
3. Run `trunk serve .`

### Issues
